Политика конфиденциальности AudioKniga.one Downloader

Последнее обновление: 07/11/2025

1. Сбор данных
Расширение НЕ собирает и НЕ передает никакие персональные данные.

2. Используемые разрешения
- storage: для сохранения настроек и прогресса скачивания
- downloads: для скачивания файлов
- cookies: для проверки авторизации на audiokniga.one
- activeTab: для получения данных с текущей вкладки
- notifications: для уведомлений о завершении скачивания

3. Хранение данных
Все данные хранятся локально на вашем устройстве.

4. Безопасность
Расширение использует только официальные API Chrome.

5. Контакты
По вопросам: info@audiokniga.one
